Customer Management System

Class project created for my client, Posh Nails. The application allows the users to create, update, and delete data for various tables. Forms can be cleared and data will display onto the forms text box when clicked from the table. The main window displays 6 buttons linked to separate windows according to the ERD and there is a glossary link on top. Also, the design was done mainly in SceneBuilder and coded to provide functionality between the application and database.


Programming: Java, SceneBuilder
Database: SQL Server


Coffee Ordering System

Class project created to demonstrate the ability to create, update, and delete orders for a coffee shop. There are 3 drop downs that allows the users to create an order and it will automatically update the subtotal, tax, and total amount. Also, users can create a new account and login/logout. For the manager view, there are 3 additional links that can be accessed separate from the employee view. Cinemagraph is displayed on the login and home page.
